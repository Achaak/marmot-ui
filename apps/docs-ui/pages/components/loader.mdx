# @pikas-ui/loader

This package contains a loader components that can be used to show a loading state.

This library is based on [React Spinners](https://www.davidhu.io/react-spinners/)

<a href="https://github.com/Achaak/pikas/blob/main/packages/ui/loader/CHANGELOG.md">
  Change Log
</a>

## Requirements

To use this package, you must install the <a href={stylesLink}>@pikas-ui/styles</a> package.

## Installation

You can install this package using npm, yarn, or pnpm.

```
npm install @pikas-ui/loader
```

```
yarn add @pikas-ui/loader
```

```
pnpm add @pikas-ui/loader
```

---

## Usage

### BarLoader

<BarLoaderExample />

```tsx
import { BarLoader } from '@pikas-ui/loader'

const Example: React.FC = () => <BarLoader />
```

#### Props

| Prop              | Description                    | Type      | Default     |
| :---------------- | :----------------------------- | :-------- | :---------- |
| `height`          | Height of the loader           | `number`  | `4`         |
| `width`           | Width of the loader            | `number`  | `100`       |
| `colorName`       | Color of the loader            | `string`  | `"primary"` |
| `colorHex`        | Color of the loader            | `string`  | -           |
| `loading`         | Show the loader                | `boolean` | `true`      |
| `speedMultiplier` | Speed multiplier of the loader | `number`  | `1`         |
