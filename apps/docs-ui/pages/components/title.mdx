# @pikas-ui/title

This package contains a title components.

<a href="https://github.com/Achaak/pikas/blob/main/packages/ui/title/CHANGELOG.md">
  Change Log
</a>

## Getting Started

You need to install the <a href={stylesLink}>@pikas-ui/styles</a> package to use this package.

### Installation

With npm:

```
npm install @pikas-ui/title
```

With yarn:

```
yarn add @pikas-ui/title
```

With pnpm:

```
pnpm add @pikas-ui/title
```

---

## Usage

### Title

<TitleExample />

```tsx
import { Title } from `@pikas-ui/title`;

const Example: React.FC = () => {
  return (
    <>
      <Title as="h1">Title H1</Title>
      <Title as="h2">Title H2</Title>
      <Title as="h3">Title H3</Title>
      <Title as="h4">Title H4</Title>
      <Title as="h5">Title H5</Title>
      <Title as="h6">Title H6</Title>
    </>
  )
}
```

#### With a custom configuration

```tsx
import { Title } from `@pikas-ui/title`;
import type { PikasCSS } from 'your_config'

const Example: React.FC = () => {
  return (
    <>
      <Title<PikasCSS> as="h1">Title H1</Title>
      <Title<PikasCSS> as="h2">Title H2</Title>
      <Title<PikasCSS> as="h3">Title H3</Title>
      <Title<PikasCSS> as="h4">Title H4</Title>
      <Title<PikasCSS> as="h5">Title H5</Title>
      <Title<PikasCSS> as="h6">Title H6</Title>
    </>
  )
}
```

#### Props

| Prop            | Description                               | Type                 | Default |
| :-------------- | :---------------------------------------- | :------------------- | :------ |
| `as`            | The HTML tag to use for the root element. | `TitleAs`            | -       |
| `variant`       | The variant to use.                       | `TitleVariant`       | -       |
| `textTransform` | The text transform to use.                | `TitleTextTransform` | -       |
| `css`           | The css to use.                           | `TitleCSS`           | -       |
| `children`      | The content of the component.             | `ReactNode`          | -       |
