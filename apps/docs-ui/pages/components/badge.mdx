# @pikas-ui/badge

This package contains a badge component.

## Getting Started

You need to install the <a href={stylesLink}>@pikas-ui/styles</a> package to use this package.

### Installation

With npm:

```
npm install @pikas-ui/badge
```

With yarn:

```
yarn add @pikas-ui/badge
```

With pnpm:

```
pnpm add @pikas-ui/badge
```

---

## Usage

### Badge

<BadgeExample />

```tsx
import { Badge } from `@pikas-ui/badge`;

const Example: React.FC = () => {
  return (
    <Badge leftIconName="bx:baguette">Bread</Badge>
  )
}
```

#### Props

Props extends `HTMLAttributes<HTMLDivElement>`.

| Prop            | Description                     | Type              | Default                |
| :-------------- | :------------------------------ | :---------------- | :--------------------- |
| `children`      | The content of the badge.       | `React.ReactNode` | -                      |
| `color`         | The color of the badge.         | `Colors`          | `"PRIMARY"`            |
| `colorHex`      | The color of the badge.         | `string`          | -                      |
| `boxShadow`     | The shadow of the badge.        | `Shadows`         | `"ELEVATION_BOTTOM_1"` |
| `borderRadius`  | The border radius of the badge. | `BorderRadius`    | `"round"`              |
| `css`           | The css of the badge.           | `CSS`             | -                      |
| `fontSize`      | The font size of the badge.     | `FontsSizes`      | `"EM-MEDIUM"`          |
| `leftIconName`  | The left icon of the badge.     | `string`          | -                      |
| `rightIconName` | The right icon of the badge.    | `string`          | -                      |
| `gap`           | The gap of the badge.           | `BadgeGap`        | `"md"`                 |
| `padding`       | The padding of the badge.       | `BadgePadding`    | `"md"`                 |
