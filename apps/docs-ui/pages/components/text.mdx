# @pikas-ui/text

This package contains a text components.

<a href="https://github.com/Achaak/pikas/blob/main/packages/ui/text/CHANGELOG.md">
  Change Log
</a>

## Requirements

To use this package, you must install the <a href={stylesLink}>@pikas-ui/styles</a> package.

## Installation

You can install this package using npm, yarn, or pnpm.

```
npm install @pikas-ui/text
```

```
yarn add @pikas-ui/text
```

```
pnpm add @pikas-ui/text
```

---

## Usage

### Label

With the `Label` component you can create a label for a form field.

<LabelExample />

```tsx
import { Label } from '@pikas-ui/text';

const Example: React.FC = () => {
  return <Label>Label</Label>;
};
```

#### Props

| Prop       | Description            | Type        | Default |
| :--------- | :--------------------- | :---------- | :------ |
| `htmlFor`  | The id of the label.   | `string`    | -       |
| `css`      | The css of the label.  | `PikasCSS`  | -       |
| `children` | The text of the label. | `ReactNode` | -       |
