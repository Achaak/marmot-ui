# @pikas-ui/table

This package contains a table component.

This library is based on [TanStack Table](https://tanstack.com/table/).

<a href="https://github.com/Achaak/pikas/blob/main/packages/ui/table/CHANGELOG.md">
  Change Log
</a>

## Getting Started

You need to install the <a href={stylesLink}>@pikas-ui/styles</a> package to use this package.

### Installation

With npm:

```
npm install @pikas-ui/table
```

With yarn:

```
yarn add @pikas-ui/table
```

With pnpm:

```
pnpm add @pikas-ui/table
```

```tsx
import { Table } from '@pikas-ui/table';

const data = [
  {
    firstName: 'tanner',
    lastName: 'linsley',
    age: 24,
    email: 'tanner@email.com',
    number: 1234567890,
  },
  {
    firstName: 'tandy',
    lastName: 'miller',
    age: 40,
    email: 'tandy@email.com',
    number: 1234567890,
  },
  {
    firstName: 'joe',
    lastName: 'dirte',
    age: 45,
    email: 'joe@email.com',
    number: 1234567890,
  },
];

const Example: React.FC = () => {
  return (
    <Table
      data={data}
      emptyMessage="No data found"
      columns={[
        {
          header: 'Age',
          accessorKey: 'age',
          id: 'age',
          cell: (props) => props.getValue<string>(),
        },
        {
          header: 'Name',
          id: 'name',
          columns: [
            {
              header: 'First Name',
              id: 'firstName',
              accessorKey: 'firstName',
            },
            {
              header: 'Last Name',
              id: 'lastName',
              accessorKey: 'lastName',
            },
          ],
        },
        {
          header: 'Data',
          id: 'data',
          columns: [
            {
              header: 'Email',
              accessorKey: 'email',
              id: 'email',
            },
            {
              header: 'Number',
              accessorKey: 'number',
              id: 'number',
            },
          ],
        },
      ]}
    />
  );
};
```

#### Props

| Prop               | Description                                 | Type                   | Default     |
| :----------------- | :------------------------------------------ | :--------------------- | :---------- |
| `data`             | The data of the table                       | `T[]`                  | -           |
| `columns`          | The columns of the table                    | `ColumnDef<T>[]`       | -           |
| `variant`          | The variant of the table                    | `TableVariant`         | `"default"` |
| `emptyMessage`     | The message to show when the table is empty | `ReactNode`            | -           |
| `hasTfoot`         | If the table has a tfoot                    | `boolean`              | `false`     |
| `pagination`       | The pagination of the table                 | `TablePaginationProps` | -           |
| `selection`        | The selection of the table                  | `TableSelection`       | -           |
| `columnOrder`      | The column order of the table               | `TableColumnOrder`     | -           |
| `columnVisibility` | The column visibility of the table          | `TableVisibility`      | -           |
| `resizing`         | The resizing of the table                   | `TableResize`          | -           |
| `sorting`          | The sorting of the table                    | `TableSorting`         | -           |
| `css`              | The css of the table                        | `TableCSS<T>`          | -           |
| `padding`          | The padding of the table                    | `TablePadding`         | -           |
| `hoverEffect`      | If the table has a hover effect             | `boolean`              | `true`      |

#### Pagination
Allows you to activate the data pagination.

| Prop                 | Description                                      | Type                          | Default |
| :------------------- | :----------------------------------------------- | :---------------------------- | :------ |
| `enabled`            | If the pagination is enabled                     | `boolean`                     | `false` |
| `state`              | The state of the pagination                      | `PaginationState`             | -       |
| `selectValue`        | The select value of the pagination               | `number[]`                    | -       |
| `onPaginationChange` | The function to call when the pagination changes | `OnChangeFn<PaginationState>` | -       |

#### Selection
Enable data selection by adding checkboxes.

| Prop                   | Description                                     | Type                            | Default |
| :--------------------- | :---------------------------------------------- | :------------------------------ | :------ |
| `enabled`              | If the selection is enabled                     | `boolean`                       | `false` |
| `state`                | The state of the selection                      | `RowSelectionState`             | -       |
| `defaultState`         | The default state of the selection              | `RowSelectionState`             | -       |
| `onRowSelectionChange` | The function to call when the selection changes | `OnChangeFn<RowSelectionState>` | -       |

#### Sorting
Allows you to activate the data sorting.

| Prop              | Description                                   | Type                       | Default |
| :---------------- | :-------------------------------------------- | :------------------------- | :------ |
| `enabled`         | If the sorting is enabled                     | `boolean`                  | `false` |
| `state`           | The state of the sorting                      | `SortingState`             | -       |
| `defaultState`    | The default state of the sorting              | `SortingState`             | -       |
| `onSortingChange` | The function to call when the sorting changes | `OnChangeFn<SortingState>` | -       |


#### Column Visibility
Allows you to activate the display or not of columns.

| Prop                 | Description                                             | Type                          | Default |
| :------------------- | :------------------------------------------------------ | :---------------------------- | :------ |
| `enabled`            | If the column visibility is enabled                     | `boolean`                     | `false` |
| `state`              | The state of the column visibility                      | `VisibilityState`             | -       |
| `onVisibilityChange` | The function to call when the column visibility changes | `OnChangeFn<VisibilityState>` | -       |



#### Column Order
Allows to activate the moving of the columns.

| Prop                  | Description                                        | Type                           | Default |
| :-------------------- | :------------------------------------------------- | :----------------------------- | :------ |
| `enabled`             | If the column order is enabled                     | `boolean`                      | `false` |
| `defaultState`        | The default state of the column order              | `ColumnOrderState`             | -       |
| `onColumnOrderChange` | The function to call when the column order changes | `OnChangeFn<ColumnOrderState>` | -       |

<Alert variant="warning">
  Column order is not enabled if you have a nested column.  
</Alert>

#### Resizing
Enables changing the width of the columns.

| Prop       | Description                                    | Type                            | Default |
| :--------- | :--------------------------------------------- | :------------------------------ | :------ |
| `enabled`  | If the resizing is enabled                     | `boolean`                       | `false` |
| `mode`     | The mode of the resizing                       | `ColumnResizeMode`              | -       |
| `onResize` | The function to call when the resizing changes | `OnChangeFn<ColumnSizingState>` | -       |
