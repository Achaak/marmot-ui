# @pikas-ui/skeleton

This package contains a skeleton component.

This library is based on [Radix Ui](https://www.radix-ui.com/).

## Getting Started

You need to install the <a href={stylesLink}>@pikas-ui/styles</a> package to use this package.

### Installation

With npm:

```
npm install @pikas-ui/skeleton
```

With yarn:

```
yarn add @pikas-ui/skeleton
```

With pnpm:

```
pnpm add @pikas-ui/skeleton
```

---

## Usage

### Skeleton

<SkeletonExample />

```tsx
import { Skeleton } from `@pikas-ui/skeleton`;

const Example: React.FC = () => {
  return (
    <Skeleton animation="pulse" width="100%" height={24} />
    <Skeleton animation="wave" width="100%" height={24} />
    <Skeleton animation={false} width="100%" height={24} />
  )
}
```

#### Props

| Prop           | Description                        | Type                    | Default   |
| :------------- | :--------------------------------- | :---------------------- | :-------- |
| `css`          | The css of the skeleton.           | `CSS`                   | -         |
| `width`        | The width of the skeleton.         | `string or number`      | -         |
| `height`       | The height of the skeleton.        | `string or number`      | -         |
| `borderRadius` | The border radius of the skeleton. | `BorderRadiusType`      | `"sm"`    |
| `animation`    | The animation of the skeleton.     | `SkeletonAnimationType` | `"pulse"` |
