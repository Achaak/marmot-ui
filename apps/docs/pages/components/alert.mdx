# @pikas-ui/alert

This package contains a alert component.

This library is based on [Radix Ui](https://www.radix-ui.com/).

## Getting Started

You need to install the <a href={stylesLink}>@pikas-ui/styles</a> package to use this package.

### Installation

With npm:

```
npm install @pikas-ui/alert
```

With yarn:

```
yarn add @pikas-ui/alert
```

With pnpm:

```
pnpm add @pikas-ui/alert
```

---

## Usage

### Alert

<AlertExample />

```tsx
import { Alert } from `@pikas-ui/alert`;

const Example: React.FC = () => {
  return (
    <Alert variant="alert">Ea Lorem fugiat aliquip irure aliquip quis mollit ullamco aliquip occaecat culpa.</Alert>
  )
}
```

### CustomAlert

<CustomAlertExample />

```tsx
import { CustomAlert } from `@pikas-ui/alert`;
import type { IconProps } from '@pikas-ui/Icons'
import { IconByName } from '@pikas-ui/Icons'

const IconExample: React.FC<IconProps> = (props) => {
  return <IconByName {...props} name="bx:baguette" />
}

const Example: React.FC = () => {
  return (
    <CustomAlert
      Icon={IconExample}
      color="WHITE"
      backgroundColor="SECONDARY"
    >
      This is an error alert.
    </CustomAlert>
  )
}
```
