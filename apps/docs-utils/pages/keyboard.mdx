## Installation

You can install this package using npm, yarn, or pnpm.

```
npm install @pikas-utils/keyboard
```

```
yarn add @pikas-utils/keyboard
```

```
pnpm add @pikas-utils/keyboard
```

---

## Usage

### useKeyPress

<UseKeyPressExample />

```tsx
import { useKeyPress } from '@pikas-utils/keyboard';

const Example: React.FC = () => {
  const isPressed = useKeyPress('Enter');

  return <div>{isPressed ? 'Enter is pressed' : 'Enter is not pressed'}</div>;
};
```
