## Usage

### getColorByExtension

Return a color based on the file extension.

Colors are based on the [Mikael Edwards](https://dribbble.com/mikaeledwards) color palette.

<GetColorByExtension />

```ts
import { getColorByExtension } from '@pikas-utils/file';

getColorByExtension('png');
// => '#df7971'
```
