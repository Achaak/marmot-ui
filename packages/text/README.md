# @pikas-ui/text

This package contains a text components.

## Getting Started

You need to install the [@pikas-ui/styles](../styles/README.md) package to use this package.

### Installation

With npm:

```
npm install @pikas-ui/text
```

With yarn:

```
yarn add @pikas-ui/text
```

With pnpm:

```
pnpm add @pikas-ui/text
```
